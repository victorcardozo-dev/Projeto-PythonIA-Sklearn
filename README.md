# Projeto-PythonIA-Sklearn
 Projeto de Previsão com Machine Learning usando o –U scikit-learn

## Funcionalidades

- Nesse projeto de previsão com Machine Learning vamos fazer uma análise de Score de cliente em um banco. 

- Vamos usar uma base de dados de 100.000 clientes para fazer o treinamento de um modelo de IA, para conseguir fazer a previsão dos do Score de Crédito dos novos clientes (se vai ser um bom cliente ou não).

## Como rodar o projeto

### Criar ambiente virtual
py -m venv .venv

### Ativar ambiente
.venv\Scripts\activate

### Instalar dependências
pip install -r requirements.txt

### Rodar aplicação
E é só executar o arquivo main.ipynb
